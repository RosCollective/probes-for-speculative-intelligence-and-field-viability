# Probes for Speculative Intelligence and Field Viability

This repository presents a set of qualitative probes and associated observational
artifacts examining speculative intelligence and field viability across
contemporary AI systems. The work is comparative and interpretive in nature and
does not provide benchmarks, rankings, or performance evaluation.
